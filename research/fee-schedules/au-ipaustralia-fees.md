# IP Australia — Fee Schedule Snapshot

- **Snapshot date:** 2026-05-16
- **Effective date of schedule:** **2024-10-01** (last general fee revision; subsequent statutory adjustments folded into the current published pages). Underlying authority: Patents Regulations 1991 (Schedule 7), Trade Marks Regulations 1995 (Schedule 9), Designs Regulations 2004 (Schedule 4).
- **Currency:** AUD (A$). All fees below are the **online ("preferred means") rate**; postal / "other approved means" surcharges noted where applicable.
- **Primary sources (canonical IP Australia pages):**
  - Patents: [Patent cost and timings](https://www.ipaustralia.gov.au/patents/timeframes-and-fees)
  - Trade marks: [Trade mark timeframes and fees](https://www.ipaustralia.gov.au/trade-marks/timeframes-and-fees)
  - Designs: [Design rights timeframes and fees](https://www.ipaustralia.gov.au/designs/timeframes-and-fees)
  - Renewals (consolidated): [How to renew my IP right](https://www.ipaustralia.gov.au/manage-my-ip/how-to-renew-my-ip-right)
  - Statutory basis: [Patents Regulations 1991 — Schedule 7](http://classic.austlii.edu.au/au/legis/cth/consol_reg/pr1991218/sch7.html)

## 1. Patents

Sources: [Patent cost and timings](https://www.ipaustralia.gov.au/patents/timeframes-and-fees)
and [How to renew my IP right](https://www.ipaustralia.gov.au/manage-my-ip/how-to-renew-my-ip-right).

### 1.1 Filing

| Action | Fee (AUD) |
|---|---|
| Provisional application | **$100** ($200 postal) |
| Standard patent application (complete) | **$400** ($600 postal) |
| International-type search request (optional) | $1,100 |
| Preliminary search and opinion (PSO) (optional) | $1,100 |

> Innovation patents were **closed to new filings on 26 August 2021**.
> Renewal fees still exist for innovation patents granted before that
> date (see §1.4).

### 1.2 Request for examination

| Action | Fee (AUD) |
|---|---|
| Examination request | **$550** |
| Excess claims fee at first report — claims 21–30 (each) | $125 per claim |
| Excess claims fee at first report — claims 31+ | $250 per claim (over 30) + $125 per claim (21–30) |
| Re-examination request | $800 |

(Excess-claim fees apply where examination is requested **on or after 1
October 2024** under the 2024 reform.)

### 1.3 Acceptance

| Action | Fee (AUD) |
|---|---|
| Standard patent acceptance (mandatory, up to 20 claims) | **$300** |
| Acceptance excess-claim fee, claims 21–30 (each, less amount already paid at first report) | $125 per claim |
| Acceptance excess-claim fee, claims 31+ | $250 per claim (over 30) + $125 per claim (21–30) |
| Amendment of specification before exam request | $250 |
| Amendment after acceptance | $250 |

The two columns are the online rate vs. the postal/cheque surcharge
rate.

| 4th | **300** | 350 |
| 5th | 315 | 365 |
| 6th | 345 | 395 |
| 7th | 380 | 430 |
| 8th | 420 | 470 |
| 9th | 465 | 515 |
| 10th | **540** | 590 |
| 11th | 645 | 695 |
| 12th | 780 | 830 |
| 13th | 945 | 995 |
| 14th | 1,140 | 1,190 |
| 15th | **1,385** | 1,435 |
| 16th | 1,675 | 1,725 |
| 17th | 2,010 | 2,060 |
| 18th | 2,390 | 2,440 |
| 19th | 2,815 | 2,865 |

## 2. Trade marks

Source: [Trade mark timeframes and fees](https://www.ipaustralia.gov.au/trade-marks/timeframes-and-fees)
and [How to renew my IP right](https://www.ipaustralia.gov.au/manage-my-ip/how-to-renew-my-ip-right).
**All fees are per class.**

| Action | Fee (AUD) |
|---|---|
| TM application — picklist (from the IP Australia 60k+ goods/services list) | **$250 per class** |
| TM application — non-picklist (custom specification) | **$400 per class** |
| Series TM application — picklist | $400 per class |
| Series TM application — non-picklist | $550 per class |
| Additional class added during exam | $450 per class |
| Postal TM application | $450 per class |
| Postal series TM application | $600 per class |
| **Renewal (10-year term)** | **$400 per class** (online) / **$450** (other means) |
| Renewal grace-period late fee | $100 per month |

> There is **no separate "registration fee"** in Australia for trade
> marks — the application fee paid up front covers initial
> registration. The next billable event is the 10-year renewal.

## 3. Designs

Source: [Design rights timeframes and fees](https://www.ipaustralia.gov.au/designs/timeframes-and-fees)
and [How to renew my IP right](https://www.ipaustralia.gov.au/manage-my-ip/how-to-renew-my-ip-right).

| Action | Fee (AUD) |
|---|---|
| Filing — first design in the application | **$200** ($400 postal) |
| Filing — each additional design in the same Locarno class | **$150** ($350 postal) |
| Further design identified at formalities check | $200 ($400 postal) |
| Request for examination (by registered owner) | **$500** |
| Examination request — third party (shared) | $250 (third party) + $250 (registered owner) |
| **Renewal (single renewal, term 5→10 years)** | **$400** online / $450 postal |
| Renewal grace-period late fee | $100 per month |

> Design term is **5 years initial + one 5-year renewal = 10 years
> max**. Unlike patents, there are no annual annuities.

## Notes — recent / scheduled changes

- **1 October 2024 fee revision** is the current effective schedule.
  Headline shifts:
  - **Years 4 & 5 patent renewal fees were held flat** ($300 / $315),
    while years 6–14 each rose by ~$10 cumulatively and years 15–19
    rose by ~$15 cumulatively per anniversary. See the IP Australia
    summary: [Fee review 2023–24](https://www.ipaustralia.gov.au/about-us/accountability-and-reporting/fee-review-2023-24)
    and the official PDF [2024 Summary table of fee changes](https://www.ipaustralia.gov.au/about-us/accountability-and-reporting/~/-/media/Project/IPA/IPAustralia/PDF/Accountability-and-reporting/CRIS/Summary-table-of-fee-changes.pdf).
  - Introduction of **excess-claim fees at the first examination
    report** for any application where exam is requested on/after 1
    October 2024 (previously, excess-claim fees were only assessed at
    acceptance). This is a meaningful prosecution-cost shift for
    high-claim-count specifications.
  - Various trade mark, design and hearing-fee adjustments rolled in at
    the same time.
- The CY 2024 revision was the first IP Australia fee revision since
  2020; the next scheduled fee review window is **2027–28** per
  IP Australia's published fee-review cadence.
- **Innovation patents:** closed to new filings on 26 August 2021;
  renewal-only schedule continues for the legacy population.
