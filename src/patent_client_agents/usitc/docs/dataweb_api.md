## DataWeb API tips

* Login via https://dataweb.usitc.gov (Login.gov + MFA) and open the API tab to copy your token.
* Set `USITC_DATAWEB_TOKEN` so requests carry `Authorization: Bearer <token>`.
* Swagger docs: https://datawebws.usitc.gov/dataweb/swagger-ui/index.html (requires auth).
* Service windows: DataWeb (and related APIs) are offline Wednesdays 5:30–8:30 PM ET
  for maintenance.
